# useful-dev-tools
Dev tools can be useful to speed up your work. 
<br>
Getting to know console methods that help you while coding can save you a lot of time.
<br>
This methods include:
<ul>
  <li>console.log()</li>
  <li>console.warn()</li>
  <li>console.error()</li>
  <li>console.info()</li>
  <li>console.assert()</li>
  <li>console.clear()</li>
  <li>console.dir()</li>
  <li>console.group()</li>
  <li>console.groupEnd()</li>
  <li>console.count()</li>
  <li>console.table()</li>
  </ul>
